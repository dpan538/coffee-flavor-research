# Security Policy

Coffee Flavor Atlas does not process user accounts, payments, private uploads,
or sensitive production data. Security still matters for dependency supply
chain issues, malicious SVGs, browser execution risks, and accidental secret
exposure.

Please do not report real vulnerabilities through public issues.

Please use GitHub's private vulnerability reporting when available. If private
reporting is not available for the repository, open a minimal public issue that
requests a secure contact path without disclosing exploit details.

No fixed response SLA is promised for this research prototype.
